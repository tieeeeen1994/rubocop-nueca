# About

This enforces custom rules for Nueca according to culture.

## Installation

Add to your Gemfile the following code.

```ruby
gem 'rubocop-nueca', require: false
```

Run `bundle install`.

# Usage

Add to your `.rubocop.yml` the following code.

```yaml
plugins:
  - rubocop-nueca
```
